## How to use this project

    * Clone the project.

    * Extract the project in your htdocs directory.

    * Set up the database: 
            => Create a database like as db_srms.sql
            => Import the database from the project folder DB/db_srms.sql 
			
	* Establish your database connection by editing includes/config.php file from the project folder.
	* For admin panel : use /admin with base url and username: admin@diu and password: admin@diu use for admin login.

     Thank You.
